# SRAS
## Installation (pour dev seulement)
- Installer node.js
- Ensuite installer electron `# npm install -g electron-prebuilt`
- Cloner le dépôt : `$ git clone https://github.com/mightstudio/SRAS.git && cd SRAS`
- Lancer le SRAS : `$ electron ./`
